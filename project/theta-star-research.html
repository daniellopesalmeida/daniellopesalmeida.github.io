Theta* Research
Overview
This project focuses on researching and implementing Theta* pathfinding, exploring how a more flexible any-angle approach can produce smoother paths than classic grid-based search.
It was developed as a gameplay programming research project in C++, with the emphasis on understanding the algorithm, integrating it into an framework, and evaluating how path quality and movement improve when agents are not restricted to rigid orthogonal navigation.
The project is a strong example of technical gameplay work because it combines algorithmic thinking, practical implementation, and clear visual feedback. It is also a good place to highlight problem solving around navigation, spatial reasoning, and making AI behavior feel more natural in motion.
