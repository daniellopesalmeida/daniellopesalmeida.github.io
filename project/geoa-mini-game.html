GEOA Mini Game
Overview
This mini game was created to showcase the use of projective geometric algebra in a simple interactive setting, connecting mathematical concepts directly to gameplay behavior.
The project was built in C++ and uses geometric algebra ideas rather than only as theory. One example is collision response, where reflection behavior can be expressed through projective geometric algebra instead of the traditional vector based movement.

static Motor Translation(float translation, const TwoBlade line)
{
    float d{ -translation / (2 * line.VNorm()) };
    return Motor{
        1,
        d * line[0],
        d * line[1],
        d * line[2],
        0, 0, 0, 0
    };
}
