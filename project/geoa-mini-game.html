GEOA Mini Game
Overview
This mini game was created to showcase the use of projective geometric algebra learned during the semester in the course of Geometric Algebra. It allowed me to apply the theories directly to gameplay behavior in a custom C++ 2D engine and using FlyFish a custom geometric algebra library.
This project demonstrates the practical application of projective geometric algebra (PGA) in the game, aiming to replace the traditional vector-based approaches with a diferent mathematical framework.
The mini game, although simple, showcases the power of PGA. By simply setting the player and enemy positions as a 3-Blade (a point) and the screen boundaries as four 1-Blade (a plane) we can use rotators, and motors to implement movement and rotation on the characters. For colisions i simply compare the distances between the points, and when below a threshold, a collision is detected. In the case of collisions to a wall, we simply use reflection of the motor 3-Blade onto the border 1-Blade, making the object appear to bounce of the wall making it a more simple implementation than traditional vector based movement.

static Motor Translation(float translation, const TwoBlade line)
{
    float d{ -translation / (2 * line.VNorm()) };
    return Motor{
        1,
        d * line[0],
        d * line[1],
        d * line[2],
        0, 0, 0, 0
    };
}
