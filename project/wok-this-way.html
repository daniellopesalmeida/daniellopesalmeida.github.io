Wok This Way
Overview
Wok This Way was developed for the Group Projects course during the final year of my bachelor's degree in Game Development. The project was created in a team of six people, with two programmers and four artists, and was developed in Unity using C#.
The goal was to create an engaging game for an existing arcade machine provided to the university by the Elaut Group. The machine's original game was built with the .NET Framework, and the player had to match colors by pressing one of the 13 buttons.
Our version used a cooking theme, where the player had to select the correct ingredients based on a recipe by matching the ingredient color to the machine's button colors. My main contributions included improving the existing machine interface, developing the game loop using two state machines, creating minigames for each ingredient, and implementing the UI.
One of the most interesting parts of the project was the added challenge of interfacing with the arcade machine while also developing the game itself. A previous group had already built a Unity interface that registered the machine input through Unity's input system, but it was not sufficient for our needs, so we extended and modified it.
One of the main issues was detecting simultaneous button presses, since the previous version would overwrite each new input. To solve this, I implemented a queue system. Another important task was creating a virtual keyboard to store the button colors and optimize input and update behavior around the IO port's bandwidth, helping us avoid flooding it and causing visual delays in the button lights.
